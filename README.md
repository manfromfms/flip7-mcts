# flip7-mcts
Monte Carlo Tree Search algorithm for the board game "Flip 7".

## Installation

Tested on Python 3.14. 

```commandline
# Make virtual environment
python -m venv .venv

# Load the venv
# Linux
source .venv/bin/activate

# Windows
# (add this command)

# Install requirements
pip install -r requirements.txt
```
